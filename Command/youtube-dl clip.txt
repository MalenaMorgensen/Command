Example using seconds:

youtube-dl --external-downloader ffmpeg --external-downloader-args "-ss 0 -to 0" "https://www.youtube.com/watch?v="

Example using timestamp:

youtube-dl --external-downloader ffmpeg --external-downloader-args "-ss 00:00:00.00 -to 00:00:00.00" "https://www.youtube.com/watch?v="
